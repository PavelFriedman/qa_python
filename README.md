# qa_python

Добавление книги

test_add_new_book_add_one_book – проверяет, что книга успешно добавляется.
test_add_new_book_invalid_name – проверяет, что книга с пустым названием или более 40 символов не добавляется.
Установка жанра книги

test_set_book_genre_valid_genre – проверяет, что жанр книги устанавливается корректно.
test_set_book_genre_invalid_genre – проверяет, что жанр не устанавливается, если он отсутствует в списке доступных.
Получение списка книг по жанру

test_get_books_with_specific_genre – проверяет, что метод возвращает книги с нужным жанром.
Работа с избранными книгами

test_add_book_in_favorites – проверяет, что книгу можно добавить в избранное.
test_delete_book_from_favorites – проверяет, что книгу можно удалить из избранного.
Книги, подходящие детям

test_get_books_for_children – проверяет, что возвращаются только книги без возрастного рейтинга.
Получение словаря книг и жанров

test_get_books_genre – проверяет, что метод возвращает корректный словарь книг и их жанров.
